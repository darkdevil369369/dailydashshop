How to edit a Daily Dash template — in Canva, Word or Google Docs

Every template is built to open in tools you already use. Here's the 3-minute version.

In Canva

  • Open the Canva link included in your download — it creates your own editable copy instantly.
  • Click any text or image to replace it. Use the top bar to change fonts, colours and sizes.
  • When you're done, hit Share → Download and pick PDF (for print) or PNG/JPG (for web and social).

In Microsoft Word

  • Open the .docx file. Click straight into any text block and type over it.
  • To change colours or spacing, use the Home and Layout ribbons — the styles are already set up for you.
  • Export a clean copy with File → Save As → PDF.

In Google Docs

  • Upload the .docx to Google Drive, then open it with Google Docs (or use the Docs link if one is included).
  • Edit as normal, then File → Download → PDF when you're finished.

Tip: keep the original file untouched and edit a copy — that way you can reuse the template again and again for free.
